# Омут Памяти

Курсовая работа — веб-приложение «Омут Памяти».

## Технологии

- **Frontend:** React, Vite
- **Backend:** FastAPI, Python

## Запуск

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
