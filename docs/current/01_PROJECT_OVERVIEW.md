# Омут памяти — обзор проекта

## Название проекта
Омут памяти

## Полная формулировка темы
Разработка информационной системы «Омут памяти» для организации учебных материалов на основе графового представления знаний

## Тип проекта
Веб-приложение / информационная система

## Назначение
Проект предназначен для хранения, структурирования и навигации по учебным материалам пользователя в едином цифровом пространстве.

## Основная проблема
Учебные материалы пользователя часто распределены между разными источниками:
- заметками;
- файлами на устройстве;
- PDF-документами;
- облачными хранилищами;
- мессенджерами;
- фотографиями и скриншотами.

Из-за этого возникают:
- разрозненность хранения;
- потеря контекста;
- сложность поиска;
- отсутствие целостной структуры;
- неудобство повторного обращения к материалам.

## Цель проекта
Разработать веб-приложение, которое обеспечивает:
- централизованное хранение учебных материалов;
- структурирование материалов по коллекциям;
- классификацию материалов через теги;
- удобную навигацию;
- визуализацию структуры в виде графа.

## Актуальный фокус
В текущей версии проект ориентирован на обучающихся и используется в учебных целях.

Система должна помогать:
- собирать учебные материалы в одном месте;
- организовывать их по темам;
- сохранять личный контекст пользователя;
- быстрее находить нужную информацию;
- визуально ориентироваться в структуре материалов.

## Основной пользователь
Учащийся:
- студент;
- школьник старших классов;
- самообучающийся пользователь.

## Объект исследования
Процесс организации учебной информации в цифровой образовательной среде.

## Предмет исследования
Методы проектирования и реализации веб-приложения, включающего механизмы хранения, структурирования и графового представления учебных материалов.

## Практический результат
Практическим результатом является прототип серверной части системы «Омут памяти», обеспечивающий:
- аутентификацию пользователя;
- работу с коллекциями;
- работу с материалами;
- работу с тегами;
- подготовку данных для графовой визуализации пользовательского пространства.

## Краткая формулировка
«Омут памяти» — это учебное веб-приложение для персональной организации учебных материалов, где пользователь хранит материалы в коллекциях, помечает их тегами и просматривает структуру материалов в виде графа.
